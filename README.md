## Installing

Setup instructions

Clone repo: `git clone git@github.com:hybridbad/spa-weather-station-vanilla.git` 

Install dependencies
`npm install`

Run local server
`npm start`

Go to to local server at port 8080:
`127.0.0.1:8080`

## Run Testing

`npm test`
